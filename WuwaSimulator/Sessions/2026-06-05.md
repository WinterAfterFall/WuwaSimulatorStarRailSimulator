# Session Notes — สิ่งที่คุยกันและสิ่งที่แก้ไปล่าสุด

---

## จุดประสงค์ของ Session นี้
ออกแบบและสร้าง **Combat Simulation System** ที่สามารถ:
1. Pre-define rotation ของตัวละครแต่ละตัวล่วงหน้า
2. จำลองลำดับ action ตาม timestamp (frame) ด้วย IndexedPriorityQueue
3. จัดการ lock system เพื่อป้องกัน action ชนกัน

---

## สิ่งที่ตกลงกันเรื่องการออกแบบ

### หน่วยเวลา
- ใช้ `frame` แทน `second` — `1s = 60 frames`

### ตัวละครบนสนาม
- มีได้ 1 ตัวต่อครั้ง (`onFieldChar`)
- หลังสลับตัวมี swap cooldown 60 frame

### Action 2 โหมด
- **Manual** → GlobalLock ON (block manual action ใหม่จาก RotationBuilder)
- **Auto** → UnitLock เท่านั้น (unit นั้น busy แต่คนอื่นทำได้)
- Rotation มีแต่ action ที่ **เริ่มต้นด้วย Manual เสมอ**
- บาง action เปลี่ยนจาก Manual → Auto กลางทางได้ (`ChangeToAuto` event)

### Sequence การ push Action
เมื่อ action ถูก execute → push ทุก sub-event พร้อมกันทีเดียว:
```
AttackActionEvent (isManual=true)
DamageEvent × N
NotificationEvent (ChangeToAuto)   ← GlobalLock OFF, UnitLock ยังอยู่
DamageEvent × M
NotificationEvent (EndAction)      ← UnitLock OFF
```

### NotificationEvent รวม signal ทุกประเภท
แทนที่จะสร้าง class แยกสำหรับ EndAction, ChangeToAuto, BuffExpired ฯลฯ
ใช้ `NotificationEvent` class เดียว + `notifyType: NotificationType` enum

## ไฟล์ที่แก้ไขใน Session นี้

### แก้ไข

| ไฟล์ | สิ่งที่เปลี่ยน |
|---|---|
| `Constants/Enum.ts` | เพิ่ม `ActionState { Free, Busy }` และ `NotificationType { ChangeToAuto, EndAction, BuffExpired, DebuffExpired }` |
| `Models/Unit.ts` | แก้ `\|\| 0` → `?? 0`, `any` → union type, เพิ่ม `isAlive()`, `setDead()`, `hasStat()`, เปลี่ยน key format ไม่ใส่ `None` ใน 2-arg case |
| `Models/AllyUnit.ts` | เพิ่ม `isOnField`, `actionState`, `isFree()`, `setBusy()`, `setFree()` |
| `Models/Combat/CombatEvent.ts` | เปลี่ยนเป็น `abstract class`, เพิ่ม `duration`, ลบ `actionName`, เปลี่ยน unit เป็น frame |
| `Simulator/CombatTimeline.ts` | เพิ่ม `onFieldChar`, `currentFrame`, `isGlobalLocked`, lock logic ใน `tick()` |
| `Simulator/RotationBuilder.ts` | เช็ค `isGlobalLocked` และ `unit.isFree()` ก่อน push, เพิ่ม `duration` parameter |
| `manualBuilder.ts` | อัปเดตเป็น frame unit, ตัวอย่าง NotificationEvent |

### สร้างใหม่

| ไฟล์ | บทบาท |
|---|---|
| `Models/Combat/ActionEvent.ts` | Abstract base — ทุก action มี `isManual`, `actionType`, `unit` |
| `Models/Combat/AttackActionEvent.ts` | Action โจมตี |
| `Models/Combat/BuffActionEvent.ts` | Action buff skill |
| `Models/Combat/DamageEvent.ts` | Damage instance |
| `Models/Combat/NotificationEvent.ts` | Signal: ChangeToAuto / EndAction / BuffExpired / DebuffExpired |
| `Simulator/CombatTimeline.ts` | จัดการ event ด้วย IPQ |
| `Simulator/RotationBuilder.ts` | Pre-define rotation |
